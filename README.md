# Modular-Wheelchair-Obstacle-Assist-System
Modular SolidWorks-based assistive wheelchair mechanism designed to help users overcome small urban obstacles (2–7 cm) using a low-cost attachable lifting structure focused on modularity and mechanical simplicity. This project is an early-stage mechanical concept and requires further structural analysis 

Project Overvie
This project is a modular assistive mechanical system designed to help manual wheelchair users overcome small urban obstacles such as curbs, uneven pavement, and low steps (approximately 2–7 cm).

The system is developed as a SolidWorks-based mechanical prototype exploring how a low-cost, attachable mechanism can enhance wheelchair mobility without fully replacing the manual propulsion system.

It focuses on a mechanical-first approach, emphasizing structural design, linkage systems, and modular integration rather than full electronic automation.
