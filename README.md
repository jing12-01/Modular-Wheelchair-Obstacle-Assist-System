# Modular Wheelchair Obstacle Assist System

## Overview
Modular SolidWorks-based assistive wheelchair mechanism designed to help users overcome small urban obstacles (2–7 cm) using a low-cost attachable lifting structure focused on modularity and mechanical simplicity.

---

## Problem
Manual wheelchair users face difficulty with:
- curbs
- uneven pavement
- small steps (2–7 cm)

---

## Design Approach
- Modular attachable mechanism
- Mechanical lifting system
- No full electronic automation
- Focus on simplicity and low cost

---

## Status
Early-stage CAD concept (not physically tested)
