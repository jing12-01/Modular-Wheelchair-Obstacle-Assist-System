# Accessibility Map Documentation

This document contains annotated accessibility maps for Zhongguancun ART PARK Plaza.

Each map represents a different spatial scale:

- LG1: Basement level circulation
- LG2: Main commercial floor circulation
- Subway connection route: external accessibility link

The maps highlight:
- Accessible routes
- Barriers and interruptions
- Elevation challenges
- Transfer points (elevators / ramps)
